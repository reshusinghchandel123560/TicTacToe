# TicTacToe
A modern and interactive Tic-Tac-Toe web application built using React, Tailwind CSS, and Framer Motion.
The game features a sleek neon-inspired arcade UI, smooth animations, intelligent AI gameplay, score tracking, and responsive design for both desktop and mobile devices.

### Features
## Two Game Modes
Player vs Player (PvP)
Player vs Computer (PvC)
## Smart AI Opponent
AI can block moves, win strategically, and prioritize optimal positions.
🎨 Modern UI Design
Neon arcade-inspired dark theme
Smooth hover effects and transitions
Fully responsive layout
## Animations
Animated X and O drawing effects
Winner reveal animations using Framer Motion
## Score Tracking
Tracks Player X wins
Tracks Player O wins
Draw counter included
## Game Controls
Restart current game
Reset scoreboard
## Responsive Design
Optimized for desktop, tablet, and mobile screens
